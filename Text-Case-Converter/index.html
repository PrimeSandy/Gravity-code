info About Text Cases

Text case conversion is essential for formatting content correctly. Our tool handles various case types instantly without sending data to any server.

Supported Formats

  • UPPERCASE: All letters are capitalized (e.g., "HELLO WORLD").
  • lowercase: All letters are lowercased (e.g., "hello world").
  • Title Case: First letter of each word is capitalized (e.g., "Hello World").
  • Sentence case: First letter of the sentence is capitalized (e.g., "Hello world.").
  • Alternating Case: Randomly capitalizes letters (e.g., "hElLo wOrLd").
